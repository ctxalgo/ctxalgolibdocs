sharpe_ratio_from_balance(balance, benchmark=None, merge=True)¶

Calculate sharpe ratio from balance ohlc. :param balance: OHLC, balance ohlc, the dates, and closes attributes represent the balance series. :param benchmark: OHLC, if not None, benchmark ohlc, the dates,

and closes attributes represent the benchmark series.
Parameters:merge – boolean, if True merge balance and benchmark into daily.
Returns:float, sharpe ratio.
static get_ohlc_with_period(ohlc, period=20, start_time=None, end_time=None)¶

Return the merged ohlc with the given period.. :param ohlc: OHLC, an ohlc. :param period: Periodicity, the period. :param start_time: datetime, the start time (inclusive) to get the information.

If None, start from the first available data point.
Parameters:end_time – datetime, the end time (inclusive) to get the information. If None, ends at the last data point.
Returns:OHLC, daily ohlc merged from ohlc.
