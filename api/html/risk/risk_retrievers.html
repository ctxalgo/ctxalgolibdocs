risk_retrievers¶

class ctxalgolib.risk.risk_retrievers.RiskRetriever¶

Bases: object

Abstract class that represents a risk metrics retriever. Descendants of current abstract class can retrieve risk metrics from different channels, such as CTX, DTS or CaiTong web portal.

retrieve()¶

Retrieve risk information for the specified product. @return: dict{string: object}, keys are risk metric names, such as ‘net_value’, values

are risk metric values. Different descendants of current class do not have to return the same set of metrics.
class ctxalgolib.risk.risk_retrievers.CaiTongRiskRetriever(account)¶

Bases: ctxalgolib.risk.risk_retrievers.RiskRetriever

Class that retrieve risk metrics from broker CaiTong

Initialize current object. @param account: string, CaiTong capital account name. @param password: string, CaiTong capital account password. @param share_holder_account: dict{string: string}, shareholder account name.

Keys are exchange names, valid values are ‘SH’ and ‘SZ’. Note, these names are case sensitive. Values are share holder account names for the corresponding exchanges. Example:{‘SH’:’A0000000’,’SZ’:‘00000000’}
@param communication_password: string, the second password needed to establish communication.
If None, it is set to be the same as password.
