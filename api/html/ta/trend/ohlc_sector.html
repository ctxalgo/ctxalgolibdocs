Bases: ctxalgolib.ta.trend.ohlc_sector.AbstractOhlcSector

Parameters:ma_calculator – object, moving average calculator. A moving average calculator returns a [float], this array represents the moving average values for each bar in the ohlc. An example moving average calculator can be found at ctxalgolib.ta.online_indicators.moving_averages.
separate(ohlc, start_bar=0)¶

Use moving average turns to calculate sections. If a moving average turns up or down, then a new section starts. The start price of the section is the close price of that bar. The start bar of the section is the index of that bar. If there is a previous non-finished section, then the end price is the close price of that bar, and the end bar is the index of that bar. If the moving average turns up, the direction of the corresponding section is 1, otherwise -1.

fast moving average up-crosses slow moving average marks the start of a up-section. fast moving average down-crosses slow moving average marks the start of a down-section. If the fast moving average up-crosses the slow moving average, the direction of the corresponding section is 1, otherwise -1.
