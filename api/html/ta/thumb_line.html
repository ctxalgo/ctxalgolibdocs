thumb_line¶

class ctxalgolib.ta.thumb_line.ThumbLineFinder¶

Bases: object

find(ohlc, bar_id, direction, n=3)¶

Find the thumb line for the bar given in bar_id. :param ohlc: OHLC, the ohlc object. :param bar_id: int, 0-based bar index. The method finds thumb line for this bar. :param direction: int, 1 means to find long direction thumb line, -1 means to find short direction thumb line.

0 means to find thumb lines for both long and short directions.
Parameters:n – int, the number of bars away (inclusive) from bar_id to identify a thumb line.
Returns:int, the 0-based bar index of the thumb line. None means that no thumb line is found.
