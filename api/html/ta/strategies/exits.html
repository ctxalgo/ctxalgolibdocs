class ctxalgolib.ta.strategies.exits.EntryStop(parameters)Â¶

Bases: ctxalgolib.ta.strategies.exits.SingleSecurityExit

Return entry stop for the next bar. If Long, return the lower price between the last low price before the transaction opening and the lowest price from the beginning of the bar when the transaction is opened until the opening time. If Short, return the higher price between the last high price before the transaction opening and the highest price from the beginning of the bar when the transaction is opened until the opening time.
