class ctxalgolib.ta.strategies.entries.SingleSecurityEntry(parameters)¶

Bases: ctxalgolib.ta.strategies.entries.EntrySignalGenerator

Entry singal generator for a single security.

Assume that the generate_signal and confirm_signal functions are called consecutively with the same ohlc serial. Otherwise, the self.last_signal and self.nonzero_signals have incorrect values.

generate_signal(ohlc, context)¶

Generate a signal given an ohlc for the next bar. If the ohlc has not been updated since the last call (checked by datetime), do not update the signal. :param ohlc: ohlc of self.sid with the same periodicity as

self.periodicity.
Parameters:context – dict, stores any other data that can be read and updated by the signal generator.

:return EntrySignal object

confirm_signal(ohlc, context)¶

Check if a signal can be confirmed with given ohlc at the execution frequency and context. If the ohlc has not been updated since the last call (checked by datetime), do not update the signal. :param ohlc: ohlc at the execution frequency. :param context: dict, stores any other data that can be read and updated

by the signal generator.

:return updated EntrySignal object.
