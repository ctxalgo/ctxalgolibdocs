indicator¶

class ctxalgolib.ta.indicator.AbstractOnlineIndicator¶

Bases: object

Class that represents an online indicator. An online indicator calculates result from right to left, and when provided with additional ohlc data, it will only calculate the additional part.

calculate(data)¶

Calculate current indicator in a right-to-left, online fashion. :param data: object, the data used for indicator calculation. It is assumed that consecutive

calls to the method will have an increasing ohlc object. So the the calculation will only need to work on the new ohlc bars.
Returns:to be defined by descendant classes.
reset()¶

Reset internal results so the next calculate will need to calculate the whole ohlc.

length()¶

Return the number of bars in current indicator. Return: int, the number of bars in current indicator.

pop(bars=1)¶

Remove the last data points. The next time when calculate is called, the calculation will start from the last not poped bar. param: bars: int, the number of bars to remove.
