extrema_indicator¶

class ctxalgolib.ta.online_indicators.extrema_indicator.ExtremaIndicator(kind, direction, period)¶

Bases: ctxalgolib.ta.indicator.AbstractOnlineIndicator

Parameters:
  • kind – string, can be ‘open’, ‘high’, ‘low’, ‘close’. It specifies the comparison base.
  • direction – int, 1 means the highest, -1 means the lowest.
  • period – int, the number of lookback bars.

reset()¶
length()¶
pop(bars=1)¶

Remove the last data points. The next time when calculate is called, the calculation will start from the last not poped bar. param: bars: int, the number of bars to remove.
