evaluate(line_sections, data1, data2)¶

This evaluator calculates performance as follows: 1. For each line section, finds the bar which is the furthest away from the line, (if both data1 and data2)

are given, use the point which is further. Let the data point used to calculate distance be p and the distance be d.
  1. Calculate the relative distance as d/p.
  2. The overall result is the average of all relative distance among all line sections.
