optimization_goals¶

class ctxalgolib.ta.parameter_tuning.optimization_goals.OptimizationGoals¶

Different optimization goals. Values:

RETURN: The return on investment. RETURN_PER_TIME_UNIT: The return on investment per time unit. RETRACE: The capital loss during a series of trading transactions.

Imagine you have capital p0 as starting capital, and you conducted a number of serial trading transactions. At the end of transaction i, you have pi money left, the retracetment is calculated as pi/p0.

WINNING_ODDS: The ratio between the number of winning transactions and losing transactions. WINNING_LOSING_RETURN_RATIO: The ratio between:

the average return on investment from winning transactions
  • the average return on investment from losing transactions
