fractal_finder¶

class ctxalgolib.ta.fractal.fractal_finder.Fractal(start_bar, level, direction, extreme_value, extreme_bar, immediate_bars, children)¶

Initialize current fractal. Arguments:

start_bar: int, the 0-based index of the first bar in current fractal. level: int, the fractal level: 0 means primitive fractal, 1 means middle level fractal, 2 means high level. direction: int, the direction of the fractal.

0 means up fractal, 1 means down fractal, 2 means direction cannot be decided (both up and down are possible).
extreme_value: float, the high (if a up fractal) or low (if a down fractal)
value of the extreme_bar in this fractal.

extreme_bar: int, the 0-based index of the highest bar (or lowest bar) in current fractal. immediate_bars: [int], list of 0-based indexes of the bars in current primitive fractal.

If current fractal is not a primitive fractal, immediate_bars will be an empty list.
children: [Fractal], the list of children fractals of current fractal.
For primitive level fractal, the children is an empty array. For middle level fractal, the children is the list of primitive fractals. For high level fractal, the children is the the list of middle level fractals. In other words, children is a recursive structure.

find(ohlc, level=1, sizes=None)¶

Find fractals from ohlc data. Parameters:

ohlc: OHLC, ohlc data for a security. level: int, 0 means primitive level, 1 means middle level, 2 means high level. sizes: [int], the size of each level of fractals.

The first element specifies how many bars a primitive fractal should contain. The second element specifies (if level=1) how many primitive fractals a middle fractal should contain. The third element specifies (if level=2) how many middle fractals a high fractal should contain.
Return:
[Fractal], the list found fractals; empty list of no fractal is found.
