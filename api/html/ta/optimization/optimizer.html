optimizer¶

class ctxalgolib.ta.optimization.optimizer.OptimizerBase¶

Bases: object

Base class for a constrained optimizer. Derived class should define the _default_options dict and the run method.

get_default_options()¶

Return the default options for current optimizer. :return: dict{string:object}, the set of default options.

run(objective, options=None)¶
Parameters:
  • objective – optimization objective object.
  • options –

    a dictinary of options for optimizer. Common keys of options dict for all optimizers: ‘max_fun_evals’ : positive integer (default np.inf), maximum number of function evaluations. ‘max_time’ : positive integer (default np.inf), maximum running time in seconds. ‘display’ : string (default ‘none’),

    ‘none’: no output ‘final’: outputs final return ‘iter’: outputs each iteration
    ‘output_history’
    : boolean (default false), when true save and return all parameter setting and their
    function evaluation outputs in a list in the stats dict (second output of the returned tuple).
Returns:

([(x, fc)], ...), a tuple with the first element being a list of optimal solutions each as a pair (x, fc) where

x: optimal parameter values, fc: output of the objective with the optimal x,

There could be other outputs depending on the derived class. If output_history is true, returns ([(x, fc)], stats).
