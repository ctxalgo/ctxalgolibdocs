class ctxalgolib.ta.optimization.objective.ObjectiveBase(param_specs, num_funs, num_cons)¶

Bases: object

Base class for objective functions in the constrained optimization problem.

The main function is eval that returns a pair of lists ([f1, f2, ...], [c1, c2, ...]). The first list includes a list of objective functions to optimize, and the second list includes a list of constraint values for constaints in the form of c_i(x) <= 0.

Parameters:
  • param_specs – ParameterVector object.
  • num_funs – number of function.
  • num_cons – number of constraints.
