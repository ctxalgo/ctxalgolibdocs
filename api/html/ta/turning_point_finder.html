turning_point_finder¶

class ctxalgolib.ta.turning_point_finder.TurningPoingFinder¶

Bases: object

static find(values, direction=0, max_count=-1)¶

Find turning points in the values. :param values: [float], the list of values. :param direction: int, 1 means to find turning-up points. -1 means to find tuning-down points.

0 means to find both turning-up and turning-down points.
Parameters:max_count – int, the maximum number of turning point to find (from right to left). -1 means to find all the turning points.
Returns:dict from int to [int]. Keys are directions, can be 1 or -1. Values are the 0-based index of the corresponding turning points in values.
