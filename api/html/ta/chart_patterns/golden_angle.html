golden_angle¶

class ctxalgolib.ta.chart_patterns.golden_angle.GoldenAngle(kind, start_bar, middle_bar, end_bar, left_loss, right_loss)¶

Class that represents a golden angle.

Initialize current. Parameters:

kind: int, 0 means upward pointing angle, 1 means downward pointing angle. start_bar: int, 0-based ohlc bar index where the angle starts. middle_bar: int, 0-based ohlc bar index where the angle reaches minimal or maximal value. end_bar: int, 0-based ohlc bar index where the angle ends. left_loss: float, the loss for left edge. right_loss: float, the loss for right edge.
class ctxalgolib.ta.chart_patterns.golden_angle.GoldenAngleFinder(linear_regressor)¶

Class that detects golden angles from OHLC data.

Initialize current. Parameters:

linear_regressor: AbstractPiecewiseLinearRegression, the piecewise linear regressor used to
find up and down sections from an OHLC time series.
default_loss_function(ohlc, start_bar, end_bar)¶

Default loss function, returning how much a period [start_bar, end_bar], both inclusive aligns as a line. Parameters:

ohlc: OHLC, time series data. start_bar: int, 0-based bar (inclusive) indicating where the line starts. end_bar: int, 0-based bar (inclusive) indicating where the line ends.
Returns:
float, a number indicating how likely the specificed period of OHLC aligns as a line. The lower the number, the better the line alignment.
find(ohlc, kind)¶

Detect golden angles from ohlc data. Parameters:

ohlc: OHLC, time series. kind: 0 means to detect upward pointing angles, 1 means to detect downward pointing angles.
Returns:
[GoldenAngle], the list of detected golden angles. Return an empty list of no angle is detected.
