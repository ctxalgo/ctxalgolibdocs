rounding_bottom¶

class ctxalgolib.ta.chart_patterns.rounding_bottom.RoundingBottomPattern(start_bar, end_bar, handle_start_bar, handle_end_bar, Ptop, Pbot, curve)¶

Class that represents rounding bottom pattern.

Initialize current pattern. Parameters:

start_bar: int, 0-based starting bar of the bowl shape. end_bar: int, 0-based end bar of the bowl shape. handle_start_bar: 0-based starting bar of the handle. -1 if no handle is found. handle_end_bar: 0-based end bar of the handle. -1 if no handle is found. Ptop, Pbot, B0, B5: float, intermediate variables used in patten identification. L1~L4, B1~B4 can be

calculated from these values.
curve: [float], the curve to visualize the cup, if handle not exist, it begin from start_bar to end_bar,
else, from start_bar to handle_end_bar

calculated from these values.

B0 = start_bar B5 = end_bar curve: [float], the curve used to visualize the half bowl pattern. curve is the array for Y-values,
