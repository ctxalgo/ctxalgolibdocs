ohlc_correlation¶

class ctxalgolib.ta.correlation.ohlc_correlation.OhlcCorrelation¶

Class to calculate different kinds of correlations among stock OHLC data.

correlation(ohlcs, data_item='close')¶

Return the correlation matrix from ohlcs. Note that the given OHLC sequences may miss values at different dates because a stock may not trade on certain dates. However, to calculate correlation, the series must contain the same number of data points. To solve the problem, first find the common trading dates among all the given ohlcs, and then use OHLC.project_by_dates to get the projected data on those common dates. If there is no value for a particular stock, then the correlation between this stock and other stocks is nan.

Use numpy.corrcoef to calculate correlations. Parameters:

ohlcs: [OHLC], list of ohlc data for multiple stocks. data_item: string, the data item from OHLC to use for correlation calculation.

Values can be “open”, “high”, “low”, “close”, “volume”. They are case insensitive.
Return:
A two-dimensional array of floats representing the correlation matrix.
correlation2(data_feed, stock_ids, data_item='close', start_time=None, end_time=None, periodicity=20)¶

Calculate OHLC correlation matrix. Parameters:

data_feed: StockDataFeed, the data feed used to load stock ohlc data.
Use StockDataFeed.trading_data2 to get OHLC objects.

stock_ids: [string], list of stock ids such as ‘sh600338’ to load. data_item: string, the data item from OHLC to use for correlation calculation.

Values can be “open”, “high”, “low”, “close”, “volume”. They are case insensitive.

start_time, end_time: datetime, the starting and ending constraints for the to-be-loaded OHLC. periodicity: Periodicity, the periodicity of the ohlc to load.

Return:
A two-dimensional array of floats representing the correlation matrix.
