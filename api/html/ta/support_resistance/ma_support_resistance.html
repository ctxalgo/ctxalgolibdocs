Default function to decide if the price at the bar position is touching the moving average. Parameters:

ohlc: OHLC, time series data. ma: [float], the moving average data from ohlc. The length of ma is equal to the length of ohlc. The

first few elements in ma is NaN .

atr: [float], the ATR value for ohlc. bar: int, 0-based index of the OHLC position. kind: int, 0 means support, 1 means suppress.

Return:
bool, True if the price at bar position is touching the moving average; False otherwise.

find_regions(ohlc, ma)¶
Find support or suppress regions from ohlc against ma.
  1. A support region consists of the longest (two or more) consecutive support points; and there is no

    price crossing the moving average too far (defined by is_crossing_within_threshold) within those points.

  2. A suppress region consists of the longest (two or more) consecutive suppress points; and there is no

    price crossing the moving average too far (defined by is_crossing_within_threshold) within those points.

Parameters:

ohlc: OHLC, time series data. ma: [float], the moving average data from ohlc. The length of ma is equal to the length of ohlc. The

first few elements in ma is NaN .
Returns:
[SupportSuppressRegion], the list of found support or suppress regions. If no region is found, return
an empty list.
