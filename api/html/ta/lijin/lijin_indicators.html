find(stock_id, ohlc, index_ohlcs, start_bar=50)Â¶

Find signals from ohlc. :param stock_id: string, id of the stock. :param ohlc: OHLC, the ohlc from which signals are to be found. :param index_ohlcs: dict{string:OHLC}, the ohlcs for indices. Used to support the TongDaXin INDEXC indicator.

Keys are index ids, values are OHLC objects for the indices..
Parameters:start_bar – int, 0-based starting bar, meaning only signals at or after this bar are to be returned.
Returns:[int], the 0-based bar index where signals occur.
Notes:
  1. The TongDaXin CAPITAL indicator is supported in StockDataFeed.stock_capital_structure with kind==1.
  2. To support TongDaXin INDEXC, assume ohlc for indices sh000001, sz399001 are stored in index_ohlcs.
