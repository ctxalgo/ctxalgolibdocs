class ctxalgolib.ta.harmonic.harmonic_pattern.AbstractHarmonicPattern(trading_direction)¶

Initialize current. Parameters:

class ctxalgolib.ta.harmonic.harmonic_pattern.AbstractHarmonicPatternDetector¶
detect(ohlc, trading_direction=None)¶

Detect harmonic patterns. Parameters:

ohlc: OHLC, ohlc data. trading_direction: TradingDirection. Specify in which trading direction to detect AB=CD patterns.

Values can be TradingDirection.LONG or TradingDirection.SHORT. If None, detect patterns in both directions.
Returns:
[AbstractHarmonicPattern], List of detected harmonic patterns.
