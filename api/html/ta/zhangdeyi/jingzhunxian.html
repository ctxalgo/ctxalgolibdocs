class ctxalgolib.ta.zhangdeyi.jingzhunxian.JingzhunLine(value, kind, touch_points)¶

Initialize current Jingzhun line (精准线) Parameters:

value: float, the horizontal position of current line. kind： int, 0 means the line is a supporting line (K bars at touch_points are above the line).

也就是说，对于supporting line, touch_points中所有K线的实体应该在精准线的上方。
1 means the line is a suppress line (K bar at touch_points are below the line).
对于suppress line, touch_points中所有K线的实体应该在精准线的下方。

touch_points: [int], an array of 0-based K bars which “touches” the line.
