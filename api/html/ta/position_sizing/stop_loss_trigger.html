stop_loss_trigger¶

class ctxalgolib.ta.position_sizing.stop_loss_trigger.StopLossTrigger¶

Class that finds when price touchs given stop loss lines.

get_trigger(data, trading_direction)¶

Get the trigger when prices touches any of the given stop loss lines. Parameters:

data: [(stop_loss_line, prices)], where stop_loss_line is of type [float], and prices is of type [float].
stop_loss_line is the time series for a stop loss line. prices is the time series for a price series. At a given time (represented as the 0-based index of price), if any price value touches its associated stop loss line, a trigger occurs.
trading_direction: TradingDirection. If the trading direction is LONG, price touching a stop loss line
means price <= stop_loss_line_value. If the trading direction is SHORT, price touching a stop loss line means price >= stop_loss_line_value.
Return: (bar_index, stop_loss_value, stop_loss_index)

bar_index: int, 0-based bar position indicating where the trigger occurs. stop_loss_value: float, the touched stop loss value. stop_loss_index: 0-based index of the input data array, indicating which stop loss line is touched by

its associated prices.

If no price touching trigger occurs, return None.
