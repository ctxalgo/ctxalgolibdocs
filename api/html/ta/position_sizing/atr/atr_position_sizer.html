get_positions_for_ohlc(ohlc, starting_bar, trading_direction)¶

Get ATR stop positions for ohlc. Parameters:

ohlc: OHLC. starting_bar: int, 0-based bar index indicating the starting ATR calculation point. trading_direction: TradingDirection, LONG or SHORT.
Returns: [float],
If trading direction is LONG, long ATR positions. The length of long_positions is the same as opens.
The first meaningful long position value is at starting_bar; all previous values are set to nan. Note that long positions are non-decreasing.
If trading direction is SHORT, short ATR positions. The length of short_positions is the same as opens.
The first meaningful short position value is at starting_bar; all previous values are set to nan. Note that short positions are non-increasing.
get_positions(opens, highs, lows, closes, starting_bar, trading_direction)¶

Calculate ATR-based positions from OHLC (open, high, low, close) data. Arguments:

opens: [float], Open prices for a time series. highs: [float], High prices for a time series. lows: [float], Low prices for time series. closes: [float], Close prices for a time series. starting_bar: int, 0-based bar index indicating the starting ATR calculation point. trading_direction: TradingDirection, LONG or SHORT.
Returns: [float],
If trading direction is LONG, long ATR positions. The length of long_positions is the same as opens.
The first meaningful long position value is at starting_bar; all previous values are set to nan. Note that long positions are non-decreasing.
If trading direction is SHORT, short ATR positions. The length of short_positions is the same as opens.
The first meaningful short position value is at starting_bar; all previous values are set to nan. Note that short positions are non-increasing.
