atr_position_sizing_parameter_tuner¶

class ctxalgolib.ta.position_sizing.atr.atr_position_sizing_parameter_tuner.AtrPositionSizingParameterTuner(objectives)¶

The AtrPositionSizer needs two parameters, time_period and delta_times. This class finds the best values for these two parameters from trading OHLC data for a security.

Initialize current optimizer.
