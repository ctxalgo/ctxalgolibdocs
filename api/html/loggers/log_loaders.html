class ctxalgolib.loggers.log_loaders.LogRecord(level, tag, dt, data)¶

Initialize current log record. A record represents a line in the output log file. :param level: string, the level of current record. :param tag: string, the tag of current record. :param dt: datetime, time of the record. :param data: dict with string keys. The data associated with current record.

get_datetime_from_string(text)¶

Get a datetime object from day, time_ and milli. Parameters:

text: string, the string representation of a datetime. day: string, a string representation of a date, in format yyyymmdd. time_: string, a string representation of a time, in format HH:MM:SS. milli: string, a strign representation of milliseconds, in format of an integer.

Return: datetime, a parsed datetime object.
