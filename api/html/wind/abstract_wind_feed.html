series(name, start_ts=None, end_ts=None, bars=None)¶

Load some time series with name and between the given range. The time series is required to have a field timestamp, which we can use to compare with start_ts and end_ts. And the timestamp field is assumed to be the first field. :param name: string, the name of the time series. :param start_ts: datetime, the starting timestamp (inclusive).

None means to start from the first row of the time series.
Parameters:
  • end_ts – datetime, the end timestamp (inclusive). None means to end at the last row of the time series.
  • bars – int, if not None, the max number of bars (at the end of the series) to return, and start_ts, end_ts will be ignored. For example, bars=-100 means to return the last 100 rows in the data. If there is not enough rows to return (max rows < 100), return the all the rows that the data has. If None, use start_ts and end_ts.
Returns:

[[object]], it is 2D array. Each inner array represents a row in the time series.
