stock_raw_base_data(start_date, end_date)¶

Return the raw base data from MySQL stock_raw_base_data table. :param start_date: date, the starting date, inclusive. :param end_date: date, the end date, inclusive. :return: numpy.ndarray. The column order is date, stock_id, open, high, low, close, volume, money, return,

vwap, split, dividend, float_shares, shares. If No data is found, return an empty numpy.ndarray.
