series(name, start_ts=None, end_ts=None, bars=None, have_suffix=False, have_header=True)¶

Load time series with name by rows. :param name: string, assume the name is a file name in self.folder, the file name ends with .csv extension.

The data file is a csv file with comma-separated columns. The first rows is a header row, which defines the field names of the time series. Use self.field_parsers and self.default_field_parser to parse the following rows in the file. If the file does not exist, return an empty array [].
Parameters:
  • start_ts – datetime, the starting timestamp (inclusive). None means to start from the first row of the time series.
  • end_ts – datetime, the end timestamp (inclusive). None means to end at the last row of the time series.
  • bars – int, if not None, the max number of bars (at the end of the series) to return, and start_ts, end_ts will be ignored. For example, bars=100 means to return the last 100 rows in the data. If there is not enough rows to return (max rows < 100), return the all the rows that the data has. If None, use start_ts and end_ts.

:param have_suffix:bool, if have file suffix. :param have_header:bool, if have column header in the file. :return: [[object]], it is 2D array. Each inner array represents a row in the time series.
