ATR止损线 :param dates: string, 开始绘制ATR止损线的日期，日期格式是yyyy-mm-dd，多个日期间用冒号隔开。如果日期不指定，则默认为标的日期的后一个交易日。 :param p: numeric, ATR期数 The min value is 1. The max value is 100. :param times: numeric, ATR倍数 The min value is 0.1. The max value is 10. :param dir: a value from the following: long, short

Parameters:
  • min_value –

    numeric, 多头止损线最低值。如果指定的话，ATR多头止损的最低值是ATR止损线和min-value相比较高的值。 该值只有在当前ATR为多头止损的情况下会使用。min-value的格式可以是如下两种： 1. 一个整数或浮点数，则表示使用该值作为min-value。 2. [ohlc]-d+, 也就是o、h、l、c四个字母之一加上一个负整数，

    表示使用之前若干根K线的开盘价(o)、最高价(h)、最低价(l)、收盘价(c)作为min-value。
  • max_value –

    numeric, 空头止损线最低值。如果指定的话，ATR空头止损的最高值是ATR止损线和max-value相比较低的值。 该值只有在当前ATR为空头止损的情况下会使用。max-value的格式可以是如下两种： 1. 一个整数或浮点数，则表示使用该值作为max-value。 2. [ohlc]-d+, 也就是o、h、l、c四个字母之一加上一个负整数，

    表示使用之前若干根K线的开盘价(o)、最高价(h)、最低价(l)、收盘价(c)作为max-value。
