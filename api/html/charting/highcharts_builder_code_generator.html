generate(indicators, class_name='HighchartsBuilder', base_classes=None, language='en')¶

Generate a Python class that is the Highcharts spec builder. During the code generation: 1. Replace all - in indicator names and parameter names with _ in method names.

For example: ‘sys-cycle’ becomes ‘sys_cycle’.

  1. Put parameters for an indicators into the generated methods according to the parameter position index.

  2. Add document to each parameter.

  3. If min_value or max_value of a parameter is not None, generate ‘assert min_value <= parameter_value’,

    or ‘assert parameter_value <= max_value’.

  4. If a parameter accepts an array of things, in url, this is a string parameter with ‘:’ as separator,

    in Python, support [string] or [date], or [datetime], or [float], or [int], as well as the original string notation.

  5. If a parameter accepts an array of array, in url, this is a string parameter with ‘::’ as separator,

    in Python, support [[string]], [[date]], [[datetime]], or [[float]], or [[int]], as well as the original string notation.

To generate string representation of an indicator parameter, use the following rules: 1. If the value equals the default value of the parameter, do not generate anything. 2. Otherwise, use the generate_parameter method to generate the string representation of the value. :param indicators: [Indicator], list of indicators. :param class_name: string, class name of generated class. :param base_classes: [string], list of base classes. :param language: string, language which is en or zh :return: string, the generated source code.
