class ctxalgolib.charting.chart_indicators.IndicatorParameter(position, name, description, min_value, max_value, default_value, kind, possible_date_range=False)¶

Class for an indicator parameter.

Initialize current indicator parameter. :param position: int, 1-based parameter position. :param name: string, name of the parameter. :param description: string, description of the parameter. :param min_value: object, can be None, the min value of the parameter. :param max_value: object, can be None, the max value of the parameter. :param default_value: object, can not be None. The default value of the parameter. :param kind: object, can be either string “numeric”, “string”, “date”, “color” or an array of pairs. :param possible_date_range: boolean, if True, then this parameter may have the form d1~d2:d3~d4.

Calculate current indicator. :param chart_generator: AbstractChartDataGenerator, the chart generator to draw indicators. :param parameters: dict from string to object. The actual parameters for current indicator. :param context: dict from string to object. The context from the chart generator.

It has the following keys:
stock_id: string, the id of the stock to display. start_time, end_time: datetime, the start and end time of the time region. cursor_date: datetime: if not None, the time of the cursor. yconfig: YaxisConfig, the y-axis config. k_yconfig: YaxisConfig, the y-axis config for the K bar area. v_yconfig: YaxisConfig, the y-axis config for the volume bar area. data_feed: AbstractDataFeed, the data feed to get data.
Returns:A dict from string to object. Keys are series category names, or values in the chart.
