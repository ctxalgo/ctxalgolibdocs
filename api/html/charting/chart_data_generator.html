get_ohlc(instrument_id, start_time, end_time, period=None)¶

Get OHLC for stock in given time region. :param instrument_id: string, id of the instrument whose ohlc is to be retrieved..

start_time, end_time: datetime, define region of the data, both inclusive. period: Periodicity, the time period, None means use the default periodicity.
:return (OHLC, Periodicity)
The first element is the retrieved ohlc data. The second element is the actual periodicity of the ohlc data. If an i happens, return (None, None)
