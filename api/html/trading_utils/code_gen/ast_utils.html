Class to generate canonical expression for an expression. A canonical expression is one that all whitespaces are normalized, so the textual representation of the expression AST can be used to identify an expression.

static get_canonical_expression(expr)¶
Return a canonical expression
AST from the given AST expression. A canonical AST removes

unnecessary whitespaces, so its text can be used directly to decide equality of two AST nodes. :param expr: Expr, An Python expression node. :return: Expr, a new expression AST which is canonical.

static find_sub_expressions(expr, sub_expressions)¶

Find sub expressions in the given expression. For example, if expr is “a + b * c” and sub_expressions is [“a”, “b”], then the result is [“a”, “b”]. :param expr: Expr, a Python expression, which is already in canonical format. :param sub_expressions: [Expr], the set of Python expressions, which are in canonical format to be

searched inside the expr.
Returns:[Expr], the set of distinct sub expressions that appear in expr.
static replace_sub_expression(expr, old_sub_expr, new_sub_expr)¶

In the expression expr, replace all occurrences of old_sub_expr with new_sub_expr. For example, if expr is “self.long_positions() > 0”, old_sub_expr is “self.long_positions()” and new_sub_expr is “v”, then the result is: “v > 0”. :param expr: Expr, a Python expression, already in canonical format. :param old_sub_expr: Expr, the expression to be replaced, already in canonical format. :param new_sub_expr: new_sub_expr: Expr, the expression to appear in result, already in canonical format. :return: Expr, the new expression with sub expression replaced.

static replace_equal_with_assignment(expr)¶

In expr, assume its structure is expr1 == expr2. Replace == operator with assignment =. For example, if expr is “a == b”, then the result expression is “a = b”. :param expr: Expr, a Python expression. :return: Assign, the expression with operators replaced.

static remove_old_expression(node)¶

In node, remove all old functions. For example, if node is “a = old(b) + 1”, then the result is “a = b + 1”. :param node: a Python AST node. :return: a Python AST node, the ast with old functions removed.

static find_read_sub_expressions(assignments, sub_exprs)¶

Given a list of assignments, assuming that those assignments have the form v = expr. And given a set of sub expressions, find all the mentioned sub expressions in the right hand sides of the assignments. For example, if the assignments are: [a = a + 1, b = a + 1, c = b + 1], and the set of sub expressions are [a, b, c], then the result is [a, b], because sub expressions “a” appears in the right hand side of the first and second assignment, and “b” appears in the right hand side of the third assignment. :param assignments: [Assign], a list of assignment AST. :param sub_exprs: [Expr], a list of sub expressions. :return: [Expr], the list of mentioned sub-expressions. This list only contains distinct sub expressions.
