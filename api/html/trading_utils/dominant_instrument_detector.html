dominant_instrument_detector¶

class ctxalgolib.trading_utils.dominant_instrument_detector.DominantInstrumentPostfix¶
static postfixes()¶

Return the set of possible dominant instrument id postfixes. ‘99 is included for backward portability. :return: set of string, dominant instrument id.

static postfix(level=0)¶

Return level-th dominant instrument postfix. :param level: int, the level, 0 means dominant, 1 means the next dominant, and so on.

A negative value means getting previous level of dominant instrument postfix.
Returns:string, the postfix.
static instrument_id(prefix, level=0)¶

Return the dominant instrument id at the given level. :param prefix: string, instrument prefix, such as IF, rb. :param level: int, the level, 0 means dominant, 1 means the next dominant, and so on. :return: string, dominant instrument id, such as IF00, IF01.

class ctxalgolib.trading_utils.dominant_instrument_detector.DominantInstrumentDetector¶

Bases: object

detect(instrument_id, level=0)¶

Is the given instrument id a dominant instrument? :param instrument_id: string instrument id. :param level: int, the level, 0 means dominant, 1 means the next dominant, and so on.

If None, it means to detect any level of dominance.
Returns:boolean, True if instrument_id is a dominant instrument. Futures whose instrument id has 99 as the delivery date part is dominant id.
is_dominant(instrument_id, level=0)¶

Same as self.detect.
