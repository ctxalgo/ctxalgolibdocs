Initialize current. :param context: zmq Context. :param address: string, the zmq queue to send responses to strategies.

If None, a default is provided.
Parameters:
  • proxy – bool, if True, send messages to a proxy.
  • send_through_thread – bool, if True, create a separate thread for message sending.
  • sleep_second – int, the number of seconds to wait after connecting to the socket. If 0, no waiting.
