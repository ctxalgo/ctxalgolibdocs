reset(data=None)¶

Reset current rule into initial states, meaning to forget current checking status. :param data: dict{string: object}, if not None, use the data to initialize current rule, which

is useful for crash recovery: when the rule checker crashes, and restarts, it loads log data from a file and use the log data to initialize all the rules.
