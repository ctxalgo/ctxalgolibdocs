class ctxalgolib.rule_checking.violations.OrderRequestRejection(product_strategy, content)¶

Bases: ctxalgolib.rule_checking.violations.AbstractRuleCheckingResult

Class that represents a rule checking violation which will cause an order request to be rejected. For example, if the number of a orders have exceeded the max allowed number of orders of a given account in a trading day, then all subsequent order requests will be rejected.

Initialize current. :param product_strategy: string, the name of the strategy. :param content: dict{string: object}, the detailed information of the rejection.
