eval(code, context)¶

Eval code in context. :param code: string, the code to evaluate. :param context: dict{string: object}, a list of variables that should be

put into the context for the code to be evaluated. Keys are variable names, values are the values of those variables.
Returns:string, if there is an exception, the exception trace; otherwise, None.
