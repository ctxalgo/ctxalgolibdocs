bar_arrival_request¶

class ctxalgolib.ohlc.bar_arrival_request.BarArrivalRequest(action, period, instrument_ids, ohlc_kind, name)¶

Bases: object

Setup an action, which will be called when all required ohlc bars have arrived. For example, if period=Periodicity.HOURLY, and instrument_ids are the list of all the traded instruments, then action will be called the next time when all hourly bars for the traded instruments have arrived. :param action: function pointer with the following four parameters:

ohlc_kind: string, the ohlc kind, can be ‘time-based’, ‘volatility-based’. period: Periodicity in case of time-based ohlc, or int/float as volatility threshold

in case of volatility-based ohlc.

name: string, the bar arrival action name, same as the name parameter specified here. ohlcs: dict{string: OHLC}, the list of ohlcs, keys are instrument ids, values are the ohlc objects.

Parameters:
  • period – Periodicity in case of time-based ohlc, or int/float as volatility threshold in case of volatility-based ohlc. This is the period for the required ohlc bars. This will be passed into the action when it is invoked.
  • instrument_ids – [string], the list of instruments whose last ohlc bar of required period must arrived before action can be invoked.
  • ohlc_kind – string, the ohlc kind, can be ‘time-based’, ‘volatility-based’. This will be passed into the action when it is invoked.
  • name – string, the bar arrival action name, used to identify different bar arrival requests. This will be passed into the action when it is invoked.
on_bar(instrument_id, bars, all_ohlcs)¶

Action to be called when there is some ohlc bars generated. :param instrument_id, string, the instrument id of the generated bar(s). :param bars: dict{string: dict{Periodicity or int/float: Bar}.

Meaning: dict{ohlc_kind: dict{Periodicity/volatility threshold: Bar}} Outer keys are ohlc kind such as ‘time-based’, ‘volatility-based’ from OhlcGeneratorConstants. Outer values are dict from Periodicity to Bar in the case of time-based ohlcs or dict from int/float to Bar in the case of volatility-based ohlcs. It may contains more than one bars: a single bar under each periodicity.
Parameters:all_ohlcs – dict{string: OHLC}, meaning dict{instrument_id: OHLC}, the ohlcs for the registered instruments at the registered period.
