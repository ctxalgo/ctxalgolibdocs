ohlc_comparer¶

class ctxalgolib.ohlc.ohlc_comparer.OhlcComparer¶

Bases: object

compare(ohlc1, ohlc2, minimal_bar_height=0.01)¶

Compare ohlc1 and ohlc2 and return the comparison report. Only need to compare opens, highs, lows, closes and volumes. Parameters:

ohlc1: OHLC, the first ohlc object. ohlc2: OHLC, the second ohlc object. minimal_bar_height: float, the minimal bar height, used in case if some ohlc bar has height 0.
Return:
dict from string to objects. Keys include:
missing_bars1: int, missing_bars2: int, common_bars: int, same_open_bars: int, same_high_bars: int, same_low_bars: int, same_close_bars: int, same_volume_bars: int, bar_distances: [float], correlation_between_opens: float, correlation_between_highs: float, correlation_between_lows: float, correlation_between_closes: float, correlation_between_volumes: float
