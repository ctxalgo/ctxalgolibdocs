pad(timestamps, values, timestamp_generator, before_values, after_values, prefer_before=True)¶

Pad time series given by timestamps and values at new timestamps generated by timestamp_generator. :param timestamps: [datetime], list of old timestamps. :param values: [[object]], multiple dimensions of values associated with timestamps. :param timestamp_generator: a generator that generates datetime series. :param before_values: [AbstractDefaultValue], list of default values used in padding for timestamps

before the first value in timestamps.
Parameters:
  • after_values – [AbstractDefaultValue], list of default values used in padding for timestamps after the last value in timestamps.
  • prefer_before – bool, whether to use before_values or after_values in case if there is not a single timestamp in timestamps.
Returns:

([datetime], [[object]], new padded time series.
