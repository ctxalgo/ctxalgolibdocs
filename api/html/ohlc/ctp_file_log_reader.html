ctp_file_log_reader¶

class ctxalgolib.ohlc.ctp_file_log_reader.CtpFileLogReader¶
load(path)¶

Load CTP file log from path. Parameters:

path: string, the log file path.
Return:
[(datetime, string, dict)], The list of logs in the log file. First element is the timestamp, second element is the data kind, third element is a dict to host the logged data.
load_ohlc(path)¶

Load ctp market data log into an OHLC object. path: string, the log file path. :return: OHLC, the loaded OHLC object.
