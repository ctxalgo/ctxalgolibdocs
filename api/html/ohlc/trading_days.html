trading_days¶

class ctxalgolib.ohlc.trading_days.TradingDays¶

Class the get days on which the stock market is trading.

trading_day(now=None, delta=0)¶

Get the trading day for the given calendar timestamp. :param now: datetime or date, calendar timestamp.

If None, current system tile is used.
Parameters:delta – int, if positive, return the next delta-th trading day relative to ts. If negative, return the previous delta-th trading day relative to ts.
Returns:date, trading day. None if now, delta gives a date which is not a trading day. For example, if now is a Sunday and delta=0.
is_after_hoilday(day)¶

to check whether a day is just after the holiday :param day: date, a date for trading day :return: boolean, a day is just after the holiday

is_trading_day(day)¶

Is the given calendar day a trading day? :param day: datetime, a calendar day. :return: bool

get_days(start_date, number_of_days)¶

Get trading days starting form starting_date (inclusive). Arguments:

@start_date: datetime, the starting date (inclusive). @number_of_days: int, the maximum number of trading days to return.
Returns:
[date] The trading days (does not require the days have already passed).
get_prior_days(end_date, number_of_days)¶

Get a certain number of trading days backward starting from end_date. Parameters:

end_date: datetime, the end date. number_of_days: int, the maximum number of trading days to get.
Returns:
[datetime], The trading days.
get_days_between(start_date, end_date)¶

Get the trading days between start_date and end_date. Parameters:

start_date, end_date: date or datetime, the start and end date (both inclusive).
Returns:
[datetime] The list of trading days.
