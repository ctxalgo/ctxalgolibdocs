class ctxalgolib.data_imports.capital_structure_importer.CapitalStructureRecord(stock_id, announced_date, action_date, reason, total, fluid_a)¶

Class that represents a capital structure record.

Initialize current record. Parameters:

stock_id: string, stock ids, such as ‘sh600001’, ‘sz000001’. announced_date: datetime, the date when current record is announced. action_date: datetime, the date when current record takes effect. reason: string, the reason for the change of capital structure. total: float, the total number of shares. fluid_a: float, the number of fluid shares in the Chinese A market.
