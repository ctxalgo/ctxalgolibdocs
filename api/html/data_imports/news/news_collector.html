news_collector¶

class ctxalgolib.data_imports.news.news_collector.AbstractNewsCollector¶
collect(category, earliest_time=None)¶

Collect news from current source and store them into database. Parameters:

category: string, the category of news that are to be collected.
Different source websites have different category.
earliest_time: datetime. All the collected news should be equal to or newer than this time.
If None, use datetime.now() - timedelta(days=3).
collect_page_content(urls, content_times, titles, categories)¶

Collect news content from current source and store them into database.

get_stock_id_from_title(title)¶

This function is to find stock’s id according to the title Parameter:

title: string, the title
Return:
string, the stock’s id; ‘’, if not find
get_urls_from_company_news(connection, start_time, end_time, source, category)¶

if url is not given, get all urls from company_news table

get_earliest_time_from_db(connection, source, category)¶

To get earliest time from database, to find the latest timestamp from the database table for those sources if no record is found, use datetime.now() - timedelta(days=3) Parameter:

connection: the database connection source: string, the source of the news category: string, the category of the news, news, report, good or index
Return:
datetime: the earliest time found
