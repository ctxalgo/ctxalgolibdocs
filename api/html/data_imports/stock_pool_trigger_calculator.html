stock_pool_trigger_calculator¶

class ctxalgolib.data_imports.stock_pool_trigger_calculator.StockPoolTriggerCalculator(connection, trigger_types)¶
calculate(pool_id=None, user_name=None)¶

Calculate trigger result for stocks in stock pool and store the results in database. Note: Since the same stock may appear in the pool for multiple times and the updated_date is different, try to minimize the number of times we load OHLC data from database. Parameters:

pool_id: int, the pool id from which stocks are calculated, if None, calculate for all pools. user_name: string, the user name for whom stocks are calculated, if None, calculate for all users.
get_trigger_date(condition, stock_id)¶

to check whether the trigger described in condition is triggered on today for the given stock :param condition: string, the condition which describe the trigger :param stock_id: string, the id of the given stock :return: date, the date of today is the trigger is enabled, otherwise, None
