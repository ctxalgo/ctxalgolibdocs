update_time(ts)¶

Update current timer with timestamp ts. This timestamp may cause the timer to trigger. If the timer is triggered, actions in self.on_time_actions will be invoked.

A timer is triggered when the first time, ts is larger than its internal next triggering timestamp. :param ts: datetime, the current timestamp. :return bool, True if the timer is triggered by timestamp ts, False otherwise.

start(ts)¶

Start current timer. Start always clears all internal time cache and start from scratch. This means that already elapsed time when the timer is stopped will be cleared. :param ts: datetime, the timestamp when current timer should be started at.

stop()¶

Stop current timer.

is_running()¶

Is current timer running? A timer is running when is it has been started, and not stopped, and the timer has not finished. A timer can finish its timing, for example, a non-repeated count-down timer with the count-down already triggered will be in a finished state, so is_running should return False even if the timer has not been stopped. :return: bool, indicates whether the timer is running.

class ctxalgolib.timer.simulated_timer.CountDownSimulatedTimer(interval, repeated=True, start_time=None, name=None)¶

Bases: ctxalgolib.timer.simulated_timer.SimulatedTimer

Class that represents a count-down timer. A count-down timer can be in repeated mode, which causes the timer to repeat its count-down after it is triggered. The timer can be in non-repeated mode, which cuases the timer to trigger at most once.

Initialize current. :param interval: timedelta, the interval for the count-down. :param repeated: bool, if True, the timer will be repeated after it has been triggered.

If False, the timer is triggered at most once.
Parameters:
  • start_time – datetime, if not None, the countdown starts from this given timestamp. If None, the countdown starts immediately after the timer starts.
  • name – string, name of the timer.

class ctxalgolib.timer.simulated_timer.AlarmSimulatedTimer(times, name=None)¶

Bases: ctxalgolib.timer.simulated_timer.SimulatedTimer

Class that represents a alarm timer, which will be triggered at given time points. This timer will not finish by itself.

Initialize current. :param times: [datetime or time or string], list of time points at which current timer should be triggered.

If an element is of type datetime, the timer will be triggered at that time point. If an element is of type time, then the timer will be triggered every day at the time point. If an element is of type string, then it must be in form of HH:MM:SS or yyyy-mm-dd HH:MM:SS. times can also be a single elemtn of datetime, time or string.
Parameters:name – string, name of the timer.
