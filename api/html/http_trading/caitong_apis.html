caitong_apis¶

class ctxalgolib.http_trading.caitong_apis.CaiTongHttpTradingApi(account, password, share_holder_account, communication_password=None)¶

Bases: ctxalgolib.http_trading.abstract_apis.AbstractHttpTradingApi

Initialize current object. @param account: string, CaiTong capital account name. @param password: string, CaiTong capital account password. @param share_holder_account: dict{string: string}, shareholder account name.

Keys are exchange names, valid values are ‘SH’ and ‘SZ’. Note, these names are case sensitive. Values are share holder account names for the corresponding exchanges. Example:{‘SH’:’A0000000’,’SZ’:‘00000000’}
@param communication_password: string, the second password needed to establish communication.
If None, it is set to be the same as password.
