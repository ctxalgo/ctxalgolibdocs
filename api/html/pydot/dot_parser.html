dot_parser¶

Graphviz’s dot language parser.

The dotparser parses graphviz files in dot and dot files and transforms them into a class representation defined by pydot.

The module needs pyparsing (tested with version 1.2.2) and pydot (tested with 0.9.10)

Author: Michael Krause <michael@krause-software.de>
