class ctxalgolib.report.abstract_strategy_report.Trade(order_id, trade_id, instrument_id, role, kind, timestamp, price, volume, position_size_percent, commission, slippage, profit, max_float_profit, max_float_loss, duration, transaction_open_time, transaction_close_time)¶

Bases: object

Class that represents a trade.

Initialize current. :param order_id: string, the order id. :param trade_id: string, the trade id. :param instrument_id: string, id of the traded instrument. :param role: string, the role type, such as OpenLong, OpenShort, CLoseLong, CloseShort. :param kind: string, the trade kind, such as market order, limit order. :param timestamp: datetime, time when the trade happens. :param price: float, price at which the trade happens. :param volume: int, the volume of the trade. Positive integer means long positions,

negative integer means short position.
Parameters:
  • position_size_percent – float, the position size percent.
  • commission – float, the commission of current trade.
  • slippage – float, the slippage of current trade.
  • profit – float, the profit of current trade. Value is meaningful when current trade is an open trade, that is, role is OpenLong or OpenShort.
  • max_float_profit – float, the max float profit. Value is meaningful when current trade is an open trade, that is, role is OpenLong or OpenShort.
  • max_float_loss – float, the max float loss. Value is meaningful when current trade is an open trade, that is, role is OpenLong or OpenShort.
  • duration – timedelta, the duration of current trade. Value is meaningful when current trade is an open trade, that is, role is OpenLong or OpenShort.
  • transaction_open_time – datetime, the transaction open time.
  • transaction_close_time – datetime, the transaction close time.

class ctxalgolib.report.abstract_strategy_report.Transaction(instrument_id, open_time, close_time, volume, profit, max_float_profit, max_float_loss, open_reason, close_reason, open_trade_id, close_trade_id, open_price, close_price, open_commission, close_commission, open_slippage, close_slippage)¶

Bases: ctxalgolib.report.abstract_strategy_report.AbstractTransaction

Class that represents a transaction. A transaction consists of an open position and close position pair. It specifies a bunch of positions that are opened and closed together.

Initialize current. :param instrument_id: string, id of the traded instrument. :param open_time: datetime, the time when the open position leg of current transaction happens. :param close_time: datetime, the time when the close position leg of current transaction happens.

Same as open_time when current is a unfinished transaction.
Parameters:volume – int, volume of the trade. Positive integer means long positions,

:param profit, float, profit of current transaction. :param max_float_profit: float, max float profit. :param max_float_loss: float, max float loss. :param open_reason: string, reason for opening the position. :param close_reason: string, reason for closing the position. :param open_trade_id: string, trade id of the open position leg. :param close_trade_id: string, trade id of the close position leg. :param open_price: float, price at which current transaction is opened. :param close_price: float, price at which current transaction is closed.

None if current is a unfinished transaction.
Parameters:
  • open_commission – float, commission occurred when current transaction is opened. None if current is a unfinished transaction.
  • close_commission – float, commission occurred when current transaction is closed. None if current is a unfinished transaction.
  • open_slippage – float, slippage occurred when current transaction is opened. None if current is a unfinished transaction.
  • close_slippage – float, slippage occurred when current transaction is closed. None if current is a unfinished transaction.

List of transactions, whose elements should be of type ctxalgolib.report.abstract_strategy_report.Transaction. Used to support rich display in IPython notebook.
